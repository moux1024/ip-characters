# 🎭 IP 角色档案库

归档虚拟角色（IP/OC/VTuber/游戏角色/动画角色等）的特征信息，包括：

- **形象特征** — 外貌、标志性穿着、辨识元素
- **语言习惯** — 口头禅、说话风格、常用词汇
- **网络评价/梗** — 社区共识、名场面、网络迷因
- **原设信息** — 官方设定、背景故事
- **流行二设** — 社区创作中最广为流传的二次设定
- **人物关系** — 与其他角色的关系、CP等

## 结构

```
characters/          # 每个角色一个 markdown 文件
  {角色名}.md
templates/           # 模板文件
  character.md      # 角色档案模板
index.md            # 角色索引 + 搜索入口
log.md              # 操作日志
```

## 使用

通过 OpenClaw `ip-profiles` skill 自动管理。

> 知识来源于网络社区整理，仅作参考用途。
